# Business & Systems Analysis (BSA) — Практические кейсы программы обучения

Тут собраны проектные и практические работы, выполненные в рамках интенсивного курса по бизнес- и системному анализу (BSA).

Этот репозиторий служит портфолио, демонстрирующим навыки работы с требованиями, моделирования процессов, проектирования системной архитектуры и декомпозиции задач.

## 📌 Структура репозитория

В данной ветке представлены **17 учебных проектов**:

* **11 индивидуальных проектов**, направленных на самостоятельную проработку ключевых хард-скиллов аналитика.
* **6 групповых проектов**, выполненных в командах.

Все директории стандартизированы и используют единое правило именования: префикс `bsa` и порядковый номер модуля (от `00` до `16`).

## 📂 Содержимое каждого проекта

Каждый проект представляет собой решение кейсов со своими бизнес-целями и детальным техническим заданием. 
Содержание каждой `bsa` папки:

* Папка `src` — финальные решения (схемы, таблицы, файлы).
* Файл `tasks.md` — задачи, которые необходимо было решить. (Они перефразированы и могут отличаться от оригинала)
* Скрин `scores.png` — финальные оценки за проекты.
