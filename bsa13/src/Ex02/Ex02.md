# Exercise 02 — Позитивные тест-кейсы для UC02

## Описание UC02 (согласно условию задачи)

**UC02 — Сформировать расписание мастера** (основное действующее лицо: Менеджер).

**Основной сценарий UC02:**

1. Менеджер открывает страницу ввода слотов мастеров на обслуживание.
2. Менеджер выбирает дату и слот, выбирает мастера из сущности «Сотрудники» с ролью «Мастер».
3. Система подтверждает, что выбранный слот не забронирован в системе клиентом.
4. Менеджер подтверждает закрепление времени за мастером.
5. Система закрепляет слот: дата, время, мастер. Статус слота устанавливается в «Свободен».

**Выбранный менеджер для тестов:**
Сотрудник `Смирнова Ольга Владимировна` (EMP007, роль «Менеджер» в сущности «Роль сотрудника»).

**Мастера для тестов:**

- Иванов Дмитрий Петрович (EMP001, роль «Мастер», статус «Работает»)
- Козлов Сергей Андреевич (EMP002, роль «Мастер», статус «Работает»)
- Соколов Алексей Иванович (EMP003, роль «Мастер», статус «Работает»)
